## Домашняя система инвентаризации

### Описание:
Приложение для учета домашних предметов и продуктов, с возможностью отслеживания количества, сроков годности и истории изменений. Позволяет организовать хранение по категориям и локациям, вести учет поставщиков, а также просматривать историю операций с предметами.

## Сущности и связи
1. **Category** – категории предметов (рекурсивные подкатегории)  
   - Поля: `id`, `name`, `parentCategory`  
   - Связь: 1:М с `Item`, рекурсивная 1:М с самой собой  

2. **Location** – места хранения  
   - Поля: `id`, `name`  
   - Связь: 1:М с `Item`  

3. **Supplier** – поставщики/магазины  
   - Поля: `id`, `name`, `contactInfo`  
   - Связь: 1:М с `Item`  

4. **Item** – предметы инвентаря  
   - Поля: `id`, `name`, `description`, `quantity`, `expiryDate`, `categoryId`, `locationId`, `supplierId`  
   - Связи: М:1 с `Category`, `Location`, `Supplier`  
   - 1:М с `InventoryHistory`  

5. **InventoryHistory** – история изменений количества  
   - Поля: `id`, `itemId`, `changeType`, `quantityChanged`, `changeDate`  
   - Связь: М:1 с `Item`

## Функциональные возможности
- CRUD для предметов, категорий, мест хранения и поставщиков  
- Добавление/удаление предметов из категорий и локаций  
- История изменения количества предметов (добавление, расход, списание)  
- Отчёты:  
  - Товары с истекающим сроком годности  
  - Остатки по категориям и локациям
